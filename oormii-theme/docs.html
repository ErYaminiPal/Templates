06. Reveal Animations

Add one of these classes to any element. The IntersectionObserver in main.js will animate it into view as the user scrolls.

/* Fade up */
<div class="reveal">…</div>

/* Fade in from left / right */
<div class="reveal-left">…</div>
<div class="reveal-right">…</div>

/* Scale up */
<div class="reveal-scale">…</div>

/* Cascade through children */
<ul class="stagger">
  <li>Item</li>
  <li>Item</li>
</ul>

/* Word-by-word headline reveal */
<h1 data-split>Wear the light within</h1>

/* 3D tilt on mouse-move */
<div data-tilt>…</div>

/* Animated counter on scroll-into-view */
<span data-count="50000" data-suffix="+">0</span>

07. Breakpoints

Mobile-first wouldn't be honest — this theme is desktop-down. The four queries cover every device.

/* Default — desktop */

@media (max-width: 1100px) {
  /* Tablet landscape */
}

@media (max-width: 800px) {
  /* Tablet portrait → mobile nav, custom cursor disabled */
}

@media (max-width: 500px) {
  /* Phone — single-column hero CTAs, denser product grid */
}

@media (hover: none) and (pointer: coarse) {
  /* Touch devices — product actions always visible, magnetic disabled */
}

08. Customisation Cookbook

The fastest paths to making this theme look like your brand.

Change the accent colour

/* assets/css/main.css — top of file */
:root {
  --gold: #b08d57;       /* warmer rose-gold */
  --gold-light: #d4b88a;
  --gold-dark: #7a5c2e;
}

Switch to navy palette

:root {
  --gold: #d4b677;
  --ink: #0a0e1a;
  --ink-soft: #131a2e;
  --slate: #2a3349;
}

Replace default images

// 1. Find every image URL
grep "images.unsplash.com" *.html

// 2. Replace with your CDN — recommended sizes:
//    Hero          2000 × 1125 (16:9)
//    Cards         800 × 1066 (3:4)
//    Blog          800 × 600 (4:3)
//    Instagram     800 × 800 (1:1)

Disable a feature

// In assets/js/main.js, find and comment out the section.
// E.g. to remove magnetic buttons:

/* ---------- MAGNETIC BUTTONS ---------- */
// document.querySelectorAll('.btn-primary, ...').forEach(...)

Add a new page

// 1. Copy any existing page as a starting point
cp about.html lookbook.html

// 2. Update title, meta description, breadcrumb, h1
// 3. Add a link in <ul class="nav-links"> on every page
// 4. Optionally add to mobile-nav and footer
